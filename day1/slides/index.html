Переменные

Переменные величины

  • Число студентов на курсе
  • Размер экрана смартфона
  • Количество лайков
  • Цена на нефть
  • Зарплата
  • ...

Характеристики переменной

  • Имя
  • Тип
  • Значение

Имя переменной

  • Короткое и непонятное:
    • x, y
    • x1, y123
    • myVar, $var__
  • Более длинное и понятное (дескриптивное):
    • counter
    • userName
    • is_user_authorized

Имя переменной

  • Может состоять из:
    • больших и маленьких латинских букв
    • цифр
    • символов $ и _
  • Не может:
    • начинаться с цифры
    • содержать пробелы

Типы переменных

  • Базовые:
    • number
    • string
    • boolean
  • Специальные:
    • undefined и null
  • Продвинутые:
    • function и object
