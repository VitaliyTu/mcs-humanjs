Упражнение «Любимый фильм»

Напишите функцию printFavoriteMovie, которая выводит название вашего любимого фильма.

Аргументы (параметры) функции

      // объявление функции
      function printHello(name) {
        console.log('Привет, ' + name);
      }
      

      // вызов функции
      printHello('Егор'); // Привет, Егор
      printHello('JavaScript'); // Привет, JavaScript

Упражнение «Несколько любимых фильмов»

Перепишите функцию printFavoriteMovie, чтобы она принимала название фильма в качестве аргумента и выводила строку «Мой любимый фильм: _название фильма_».

В случае, если название фильма не передано, должна выводиться строка «У меня нет любимого фильма :-(».

Упражнение «Много любимых фильмов»

Перепишите функцию printFavoriteMovie, чтобы она принимала массив с фильмами в качестве аргумента, и затем выводила их названия через запятую.

Возвращаемое значение

      function sum(a, b) {
        return a + b; // функция возвращает результат
      }
      

      var s = sum(5, 10); // вызов функции с параметрами
      console.log(s); // 15

Упражнение «Сумма чисел в массиве»

Напишите функцию arraySum, которая принимает массив с числами в качестве параметра, и возвращает их сумму.

Пример использования:

      var s1 = arraySum([1, 2, 3]);
      var s2 = arraySum([10, 20, -5]);
      console.log(s1); // 6
      console.log(s2); // 25
